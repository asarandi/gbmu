# gbmu



#### description:

work in progress; gameboy emulator in c;


#### building:

`apt install libsdl2-dev`

`make`


#### usage:

`./gbmu <gameboy_rom_file.gb>`


#### controls:

|gameboy|gbmu|
|-|-|
|directional pad|arrow keys|
|a|z|
|b|x|
|start|return|
|select|right shift|
