# gbmu

## description

work in progress; gameboy emulator in c;


## building

### on linux

```
    apt install libsdl2-dev
    make
```

### on macOS

```
    brew install sdl2
    make
```

## usage

```
    ./gbmu rom_file.gb
```

## controls

|gameboy|gbmu|
|-|-|
|d-pad|arrow keys|
|b|z|
|a|x|
|start|return|
|select|right shift|
